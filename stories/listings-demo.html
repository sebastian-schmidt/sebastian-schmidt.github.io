Listings Demo

Nikola intends to let you show code easily via listings:

`hello.py <link://listing/listings\hello.py>`_

#!/usr/bin/python

import sys


def hello(name='world'):
    greeting = "hello " + name
    print(greeting)

if __name__ == "__main__":
    hello(*sys.argv[1:])
